Como Surgiu a Internet

A internet surgiu a partir de um projeto militar nos Estados Unidos durante a Guerra Fria. Nos anos 1960, o Departamento de Defesa dos EUA queria criar uma rede de comunicação que pudesse resistir a ataques e garantir a troca de informações mesmo se parte da infraestrutura fosse destruída. Assim, nasceu a ARPANET (Advanced Research Projects Agency Network).

A ARPANET conectava universidades e instituições de pesquisa, permitindo a troca de dados e a comunicação entre cientistas. O primeiro link foi estabelecido em 1969 entre a Universidade da Califórnia em Los Angeles (UCLA) e o Instituto de Pesquisa de Stanford.

Nos anos 1970, surgiram os protocolos de comunicação TCP/IP, desenvolvidos por Vint Cerf e Bob Kahn, que permitiram a padronização e expansão da rede. Em 1983, a ARPANET adotou oficialmente o TCP/IP, marcando o início da internet como a conhecemos hoje.

Na década de 1980, a rede se expandiu além das universidades e instituições de pesquisa, alcançando empresas e indivíduos. O surgimento da World Wide Web (WWW) em 1991, criada por Tim Berners-Lee, revolucionou o acesso à informação, permitindo a navegação por meio de hiperlinks e a criação de páginas web.

A partir daí, a internet cresceu exponencialmente, tornando-se uma ferramenta essencial para comunicação, trabalho, educação e entretenimento em todo o mundo.

Como Funcionam as Tags em HTML

Tags HTML são instruções que informam ao navegador como formatar o texto. Você pode usar tags para definir itálico, quebras de linha, objetos, bullet points e muito mais. Essas tags estão presentes no HTML (Hypertext Markup Language) de todas as páginas da internet.

Como as páginas leem as tags HTML?

Os servidores leem o código HTML para entender e renderizar o conteúdo corretamente. Essa leitura é feita de cima para baixo, da mesma forma que você está lendo este artigo agora. Você pode usar quantas tags quiser para formatar o conteúdo. No entanto, existem algumas regras essenciais sobre tags HTML que você precisará seguir. Uma tag HTML deve conter três partes: Uma tag de abertura — começando com um símbolo <>; Conteúdo — instruções curtas para exibir o elemento na página; Uma tag de fechamento — terminando com um símbolo . Algumas tags HTML podem ser abertas. Isso significa que elas não precisam ser fechadas com . Essas tags são normalmente usadas para metadados ou quebras de linha.

Tags e Conteúdo:

Cada tag define uma parte específica do conteúdo. Por exemplo:

<h1> a <h6>: Tags de cabeçalho, usadas para títulos e subtítulos.

<p>: Define um parágrafo.

<a>: Cria um link. Exemplo: <a href="https://www.exemplo.com">Clique aqui</a>.

<img>: Insere uma imagem. Exemplo: <img src="imagem.jpg" alt="Descrição da imagem">.

Atributos:

As tags podem ter atributos que fornecem informações adicionais. Os atributos são escritos na tag de abertura e têm a forma de pares nome-valor. Por exemplo, na tag de link <a>, o atributo href especifica o destino do link:

<a href="https://www.exemplo.com">Visite o Exemplo</a>

Funcionamento:

1. Escrever o HTML: Cria-se um arquivo HTML com a estrutura e os elementos desejados.

2. Abrir no Navegador: O arquivo HTML é aberto em um navegador web.

3. Renderização: O navegador lê o código HTML e renderiza o conteúdo na tela de acordo com as tags e atributos especificados.
